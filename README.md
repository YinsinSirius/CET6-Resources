### 一些特别说明

- 21 22年除了6月、12月，还有3月、9月的真题。这是因为当年疫情延误了一部分学校正常考试，教育局在3月、9月安排了统一补考
- 每一次四六级都有三套卷子。每套卷子只有听力是【素材相同，选项顺序不同】，阅读、作文、翻译都是一套一版
- 核对答案时请参照**选项**，<u>不要看ABCD</u>
